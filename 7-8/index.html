💡 엔트리 인공지능 모델 학습 방법

안내된 단계에 맞추어 인공지능에게 우리의 목소리를 학습시켜 보세요.

1
엔트리 로그인 및 작품 만들기 시작
엔트리(playentry.org) 로그인 후 상단 만들기에서 작품 만들기 화면을 엽니다.
2
모델 학습하기 메뉴 이동
좌측 블록 탭에서 인공지능을 누르고 초록색 모델 학습하기 버튼을 클릭합니다.
3
분류: 소리 모델 생성
새로 만들기 창을 실행하여 마이크 모양의 소리 분류 카드를 고르고 학습하기를 시작합니다.
4
클래스 단어 변경 및 목소리 녹음
클래스 이름을 각각 [앞]과 [뒤]로 변경한 뒤, 마이크를 이용해 나만의 명료한 음성을 여러 차례 녹음해 넣습니다.
5
인공지능 학습 실행 및 테스트
우측 하단의 모델 학습하기를 눌러 공부를 마치고, 마이크에 대고 말을 해 보며 정확히 구분해 내는지 검증합니다.
🛠️ 블록 조립 힌트!
학습을 마친 뒤 코딩 창으로 돌아와 [인공지능] ➡️ [인공지능 블록 불러오기] ➡️ [오디오 감지] 블록을 함께 조립하여 활용해 보세요!

📢 말하고 듣는 AI의 유연성 키우기 (유사 단어 설계)

인공지능은 학습하지 않은 단어는 인식하지 못해요. 우리가 말하는 다양한 단어들을 미리 예측하여 동물이 "앞으로" 가거나 "뒤로" 가도록 단어 사전을 설계해 보세요!

🏃 '앞'과 비슷한 단어 입력

예시: 앞쪽, 전진, 고, 앞으로가, 출발 등

기록된 '앞' 유사어가 없습니다.
🚶 '뒤'와 비슷한 단어 입력

예시: 뒤로, 후진, 빽, 뒤쪽으로, 후퇴 등

기록된 '뒤' 유사어가 없습니다.
