# DIGIMON · 데이터로 만나는 반려동물 사회

2026 티처스랩 교사연구회 5기 수업 활동지 모음.

## 차시 구성

| 차시 | 주제 | 활동지 |
|---|---|---|
| 1~2 | 데이터로 만나는 반려동물 사회 (문제 마주보기) | [열기](1-2/) |
| 3~4 | 나만의 반려동물 만들기 (프롬프트) | 개발 예정 |
| 5~6 | 먹이 주기 (이미지 분류) | [열기](5-6/) |
| 7~8 | 사이버 동물 키우기 (음성 인식) | 개발 예정 |
| 9   | 우리 반 캠페인 (캔바) | 개발 예정 |

## 사용 기술

- Tailwind CSS · FontAwesome · Jua 폰트
- Chart.js (데이터 시각화)
- html2canvas (활동 결과 사진 저장)
- localStorage (자동 저장)

## 데이터셋

각 활동지의 데이터셋은 해당 차시 폴더 안 `data/` 디렉토리에 있습니다. 자세한 출처는 각 폴더의 `data/README.md` 참고.
